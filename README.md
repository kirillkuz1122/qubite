# Qubite - Универсальная Платформа для Соревнований и Тестирований

## Обзор проекта

Qubite — это гибкая веб-платформа (аналог Quizizz/Kahoot), предназначенная для проведения школьных олимпиад, "мясорубок", домашних заданий и вебинаров. Платформа поддерживает различные форматы вопросов, включая тесты, открытые ответы и написание кода.

## Проблематика

- Отсутствие удобных инструментов для автоматизации сложных школьных соревнований.
- Муторность ручной проверки ответов на бумаге.
- Необходимость универсального комбайна для разных учебных дисциплин.

## Основной функционал (MVP)

### 1. Ролевая модель

- **Участник:** Решает задачи, участвует в турнирах (лично или в составе команды), видит лидерборды.
- **Организатор (Учитель):** Создает турниры, управляет банком заданий, настраивает правила (таймеры, видимость результатов), модерирует апелляции.

### 2. Конструктор Соревнований

- **Типы заданий:**
    - Выбор варианта (A/B/C/D).
    - Краткий текстовый ответ.
    - Развернутый ответ (проверяется вручную или ИИ-ассистентом).
    - Программный код (Python/JS и др.) с проверкой через тесты.
- **Настройки турнира:**
    - Ограничение времени (на всё соревнование или по-вопросно).
    - Видимость результатов (сразу или после финиша).
    - Тип участия (индивидуально/командно).
    - Уровни доступа (открытый по ссылке, по пин-коду, закрытый по списку).

### 3. Техническая архитектура

- **Frontend:** SPA (React/Next.js/Vite), адаптивный дизайн (упор на 320px+ для мобильных).
- **Backend:** Node.js/Python (в разработке).
- **Авторизация:** Email/Пароль + OAuth (Google, VK).
- **Интеграции:** AI-модуль для автоматической оценки сложных ответов, экспорт/импорт Excel.

### 4. Социальные функции

- **Команды:** Пользователи могут объединяться в команды для участия в групповых зачетах.
- **Лидерборды:** Живой рейтинг участников во время и после соревнований.
- **Апелляции:** Система тикетов для разбора спорных моментов в вопросах.

## Стек технологий

- **Core:** HTML, CSS, JS.
- **Styling:** Vanilla CSS (современные переменные, гриды, анимации).
- **UI:** Кастомная дизайн-система (Premium Dark Mode, Glassmorphism).
