# Front-end UI Rules

**КРИТИЧЕСКОЕ ПРАВИЛО:**

1. Перед написанием любого UI-кода **всегда** запрашивай актуальные стили, дизайн и ассеты через Pixso/Stitch MCP.
2. **СТРОГИЙ ЗАПРЕТ**: Никогда не используй случайные цвета, шрифты, размеры или отступы "из головы". Все значения (отступы, цвета, etc.) должны браться строго из дизайн-системы или Pixso MCP.
