# Anti-Hallucination Protocol

**КРИТИЧЕСКОЕ ПРАВИЛО:**
Если промпт пользователя слишком короткий, допускает двоякое толкование, или тебе не хватает контекста для 100% уверенности:

1. **СТОП.** Не пиши код, не вноси изменения и не выполняй поставленную задачу.
2. Сначала задай пользователю нумерованный список уточняющих вопросов.
3. Начинай писать/менять код или выполнять задачу только тогда, когда будешь полностью уверен. Если уверенности нет — продолжай задавать вопросы до полного прояснения ситуации.
